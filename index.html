Que7 Write a function that accepts parameter n and returns factorial of n 

            
Answer

            function factorial(n){
                let prod=1;
                 for (let i=1; i<=n; i++){
                    prod=prod*i;
                 }  
                 console.log(prod) 
            };
            factorial(10);
